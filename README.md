# AutoFill Pro ⚡

افزونه هوشمند کروم برای پر کردن خودکار فرم‌های کاریابی

## ویژگی‌ها

- 📄 **آپلود رزومه** — PDF, DOCX, TXT, JPG, PNG
- 🤖 **تشخیص خودکار فیلدها** — با استفاده از نام، ID، placeholder
- 🌐 **پشتیبانی فارسی + آلمانی + انگلیسی** — هوشمند در شناسایی زبان
- 🚀 **پر کردن با یک کلیک** — دکمه پر کردن در هر صفحه
- 📎 **آپلود خودکار فایل** — رزومه و عکس پروفایل
- 🔒 **امن** — همه چیز لوکال ذخیره میشه، سروری نداره
- 🎨 **رابط کاربری زیبا** — تم تاریک مدرن

## نصب

1. پوشه `autofill-extension` رو دانلود کنید
2. در کروم برید به `chrome://extensions`
3. **Developer mode** رو فعال کنید
4. **Load unpacked** رو بزنید
5. پوشه `autofill-extension` رو انتخاب کنید

## استفاده

1. روی آیکون افزونه کلیک کنید
2. رزومه کلاینت رو آپلود کنید (PDF/DOCX/متن)
3. اطلاعات استخراج شده رو بررسی کنید
4. به سایت کاریابی برید
5. روی **🚀 پر کردن فرم** کلیک کنید

## ساختار پروژه

```
autofill-extension/
├── manifest.json              # Manifest V3
├── icons/
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
├── popup/
│   ├── popup.html             # رابط کاربری
│   ├── popup.css              # استایل
│   └── popup.js               # منطق آپلود
├── content/
│   └── content.js             # تشخیص + پر کردن فرم
└── background/
    └── service-worker.js      # سرویس پس‌زمینه
```
